# MangaView
마나모아 크롤러

used in [MangaViewAndroid](https://github.com/junheah/MangaViewAndroid)

windows executable binaries are wrapped using [launch4j](http://launch4j.sourceforge.net/)

## License: ##
[MIT License](LICENSE)

## 기능: ##
- 만화 검색 및 다운로드
- url로 만화 다운로드
- 최신만화 댓글 봇(베타)

## 요구사항: ##
- jre 9 이상
